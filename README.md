# StudySprint

## Overview
StudySprint is a full-stack web application designed to help students stay focused using a study timer and track their study sessions.

## Features
- Timer with start, pause, and reset functionality  
- Save study sessions  
- View all sessions  
- Edit and delete sessions  
- Search sessions  
- User login and registration  

## Technologies Used
- HTML, CSS, JavaScript  
- Node.js, Express  
- MongoDB  

## How to Run

1. Start MongoDB  
2. Run backend:
   node server.js  
3. Open index.html in browser
